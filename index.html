What’s Included?

  • Leaflet 0.4.2 - Mapping library
  • Leaflet.fullscreen - HTML5 full screen button
  • leaflet-hash - Adds lat, lon and zoom level to URL
  • Leaflet Control Bing Geocoder - Search bar
  • jQuery 1.7.2 - Because everything includes jQuery
